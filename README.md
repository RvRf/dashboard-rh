 Análise de Dados - RH

 Contexto
- Projeto desenvolvido para fins de estudo e prática em análise de dados, utilizando Power BI e Excel.

 Objetivo
- Analisar dados do RH para identificar total de funcionários, funcionários por gênero e despesas.

 Ferramentas Utilizadas
- Power BI
- Microsoft Excel

 KPIs
- Total de funcionários
- Gastos com salários
- Faltas de funcionários

 Análises Realizadas
- Comparação de salários por departamentos
- Análise de faltas por filial
- Identificação de funcionários com maiores salários
- Análise de idade médio dos funcionários
